measure

measure library Null safety

A collection of measurement functions.

All measurements are in meters. All measurements are in the EPSG:4326 coordinate system.

Functions

bearing(Point from, Point to) → double
Computes the bearing between two Points.
distance(Point from, Point to) → double
Computes the distance between two Points in meters.
length(Feature line) → double
Calculates the length of a LineString or MultiLineString, in meters.
